Tally Recording

Jar file: reesv11.jar

Java Doc: doc/index

Java Source: reesSource.zip

Options

  1. Configure services to tally
  2. Tally the use of services
  3. Report on the use of all services

Features

  1. User configurable service types(Add,Remove,Modify,Disable Entry)
  2. Phone and Walkin service separated
  3. Reports for Day, Week, Month, Quarter
  4. Printable Reports
  5. Spread Sheet Compatable Comma Delimited Reports

Known bugs

  1. Disabling reportable items does not appear to be working
