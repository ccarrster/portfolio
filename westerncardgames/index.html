Kraplow!

Java/Javascript/HTML Source Multi-player, Multi-game: https://github.com/ccarrster/kraplow

URL: Play now(4-7 player game)

Game options

  1. 4 - 7 Players
  2. Creating a game
  3. Joining a game
  4. Starting a game
  5. Drawing cards
  6. Playing a card
  7. Discarding cards
  8. Responding to cards
  9. Using special abilities

Features

  1. Multi-player browser-based western card game
  2. Custom artwork
  3. Site wide chat
  4. Playable cards
  5. Potential targets of cards highlight on mouse over
  6. Highlights your player
  7. Highlights the active player
  8. Images have alt text
  9. Images have titles
  10. Multiple games at once
  11. Information/Title screen
  12. Game Lobby
  13. Command
  14. Draw/Discard
  15. Game State
  16. Game log
  17. Image Legend
